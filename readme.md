## Apache+Tomcat6 Web Container Sample

# 起動
```
docker-compose up
```

# デーモン起動
```
docker-compose up -d
```

# imageを構築
```
docker-compose up --build
```

# 関連の破棄
```
docker-compose down --rmi all --volumes
```
